Bem vindo ao Edools

Somos a plataforma EAD perfeita para você.

Seja vendendo cursos online ou treinando colaboradores, nossa plataforma tem tudo o que você precisa para gerenciar e comercializar seus cursos online de forma completa e profissional.

Nossa solução alia simplicidade, o suficiente para que qualquer pessoa consiga criar uma escola online em alguns minutos e já comece a vender, e robustez, permitindo que grandes instituições adotem tanto para comercialização de cursos online como para treinamento de colaboradores.

Neste site você encontrará todas as informações necessárias para usar nossa API e nosso sistema de temas.

Personalizando Temas

Com nossos temas cada cliente pode construir o seu ambiente de ensino online com identidade visual própria, tendo como base todo o poder tecnológico que o Edools oferece.

Chega de ambientes de ensino feios e entediantes, quebramos os paradigmas e elevamos o ensino online a um novo patamar, com plataformas bonitas, elegantes e apaixonantes.

Com um Tema Edools é possível:

  • Personalizar a aparência da sua escola, através de HTML/CSS;
  • Alterar a dinâmica da página, adicionando sliders, scrolls, animações e etc.
  • Utilizar seus próprios scripts ou plugins jQuery;
  • Moldar os dados a serem mostrados, para que se encaixem melhor ao seu modelo de negócio.

Trabalhando com API

Com a nossa API você será capaz de integrar qualquer sistema externo ao Edools, realizar migrações de dados de plataformas antigas de maneira simples e direta. Poderá construir aplicativos para extender uma funcionalidade existente ou até mesmo criar novas funcionalidades com sua lógica própria.

Através da API você tem toda a flexibilidade necessária para adaptar o Edools às suas regras de negócio, tendo uma plataforma de ensino online que se encaixe perfeitamente às suas necessidades.
