V1::SessionsController

Sessions

create

URL

POST /users/sign_in

Parâmetros obrigatórios

Nome Tipo Descrição
user Hash

The User to sign in
user[email] String

The user's email. Can pass login instead
user[login] String

The user's username. Can pass email instead
user[password] String

The user's password
realm String

The user's realm
realm[type] String

The realm type. Can be School or Organization
realm[id] Integer

The realm id

Objeto resposta

User

If sign in is ok it will return the user, if not will return unauthorized

destroy

URL

DELETE /users/sign_out

Objeto resposta

ApiKey

The destroyed user's ApiKey
