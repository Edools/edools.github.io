show

URL

GET /certificate_layouts/:id

Parâmetros obrigatórios

Nome Tipo Descrição
id Integer

Identifier the certificate_layout

CertificateLayout

create

URL

POST /schools/:school_id/certificate_layouts

Parâmetros obrigatórios

Nome Tipo Descrição
certificate_layout Hash

The school certificate_layout to create
certificate_layout[logo_url] String

Logo image url of the certificate_layout
certificate_layout[bg_url] String

Background image url of the certificate_layout
certificate_layout[signature_url] String

Signature image url of the certificate_layout
certificate_layout[signature_text] String

Signature text of the certificate_layout
certificate_layout[template] String

Template of the certificate_layout

URL

PUT/PATCH /certificate_layout/:id

Parâmetros obrigatórios

Nome Tipo Descrição
certificate_layout Hash

The certificate_layout to update
certificate_layout[name] String

Name of the certificate_layout
